# 2DPlatFormerGame
HCI project in 2022 Fall 
--------------

## Development Environment
- Language: C#
- Framework: Unity
    - Unity Version: 2021.3.9f1 
    - Server: Photon Pun2

## Project Overview
- Multiplayer competitive platformer game
- Players race to end for points
- 2D side-scroller
- Built in lobby System
